# 【唯二主義】个人博客-前台部分
## 技术栈
- React SRR
  - React
  - Next
  - Rudex
  - React-Rudex
  - Rudex-Thunk
  - Rudex-Persist
- Ant Design
- Axios
## 预览地址
[http://vear.vip/](http://vear.vip/)
### 安装依赖
```
npm install
```
### 运行
```
npm run start
```
### 打包
```
npm run build
```
